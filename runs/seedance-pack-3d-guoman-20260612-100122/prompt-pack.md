# Canvas Prompt Pack

这个包只给 Canvas 手动生成使用，不包含图片、视频或媒体文件。

## 使用顺序

1. 导入 canvas-project.zip。
2. 本包只生成基础资产：3D国漫整体风格、许家大堂、4个人物三视图、3个核心道具。
3. 左侧资料节点不要生成；点击右侧图片节点生成。
4. 生成满意后锁定这些资产；下一步再做每5条=15s的Seedance逐条视频文本投喂包。
